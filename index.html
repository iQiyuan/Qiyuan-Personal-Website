  • Vid2Arm: Enabling Robot Arm to Learn Human Motion from Videos

    Learning-Based Control

    Motion imitation is a challenging task in robotics and computer vision, typically requiring expert data obtained through MoCap devices. However, acquiring such data is resource-intensive in terms of financial resources, manpower, and time. Our project addresses this challenge by proposing a novel model that simplifies Motion Imitation into a prediction problem of joint angle values in reinforcement learning.

  • Multi-Agent Task Allocation for collaborative Objects Retrieval

    Navigation & Planningn

    Traditional VSP-based task allocation methods often lack adaptability to dynamic changes in tasks and environments. To address this limitation, our model introduces a task allocation algorithm that leverages vehicle-centered Voronoi region partitioning. By dynamically adjusting the weights of Voronoi cells, this method adapts to environmental and task changes in real time.

  • Fully Autonomous Indoor Navigatiom for iScan2BIM

    Navigation & Planning

    Tackled key challenges in indoor navigation by integrating high-precision LiDAR-based SLAM (LIO-SAM) for robust localization and real-time mapping in GPS-denied environments. Employed a hybrid A global planner* with DWA (Dynamic Window Approach) for efficient local path planning and dynamic obstacle avoidance in complex indoor environments.
